Rozcestník pro značku **Sázky DOBRÁ VĚC**, který slouží jako sběrný web pro příchozí ze sociálních sítí.

## Jak začít?
Požíváme **Yarn**

```bash
yarn install

yarn dev
```

## Použité technologie
- Next.js
- React
- Typescript
- Tailwind CSS
